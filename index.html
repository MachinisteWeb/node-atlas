Utilisation CLI

Installer NodeAtlas

> npm install -g node-atlas

Créer un site test

> nodeatlas --init

Lancer le site

> nodeatlas --browse

Utilisation API

Installer NodeAtlas

> npm install node-atlas

Créer un script

server.jsvar NA = require("node-atlas"),
    website = NA();
website.config({
    "init": true
}).afterInitProject(function () {
    website.run({
        "browse": true
    });
}).init();

Lancer le site

> node server.js
