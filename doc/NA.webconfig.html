Namespace: webconfig

NA. webconfig

Contain Webconfig file to JSON format.

Members

staticNA.webconfig.assetsRelativePathString

Folder for public file like image, CSS or JS.
Default Value:
  • "assets/".

staticNA.webconfig.bundlesObject

Contain bundle configuration for CSS and JS minification to JSON format.
Default Value:
  • The webconfig's property `bundles`.

staticNA.webconfig.componentsRelativePathString

Default folder for template engine include file or controller components.
Default Value:
  • "components/".

staticNA.webconfig.controllersRelativePathString

Controller folder for Back-end.
Default Value:
  • "controllers/".

staticNA.webconfig.generatesRelativePathString

HTML asset generation Folder.
Default Value:
  • "generates/".

staticNA.webconfig.httpHostnameString

Server listening hostname by http (for reverse proxy).
Default Value:
  • "localhost", or `process.env.IP_ADDRESS` if setted, or the webconfig's property `httpHostname`.

staticNA.webconfig.httpPortString

Server listening port.
Default Value:
  • 80, or `process.env.PORT` if setted, or the webconfig's property `httpPort`.

staticNA.webconfig.routesObject

Contain all routes to JSON format.
Default Value:
  • The webconfig's property `routes`.

staticNA.webconfig.templatesRelativePathString

Template folder for template engine.
Default Value:
  • "templates/".

staticNA.webconfig.urlHostnameString

Url access hostname by http (for reverse proxy).
Default Value:
  • `NA.webconfig.httpHostname`.

staticNA.webconfig.urlPortString

Url access port.
Default Value:
  • `NA.webconfig.httpPort`.

staticNA.webconfig.urlRelativeSubPathString

Adding subfolder to original url.
Default Value:
  • Empty.
Example
// If `NA.webconfig.urlRelativeSubPath` is setted to "example/"
// Website will run by default to « http://localhost/example/ »

staticNA.webconfig.urlWithoutFileNameString

Website http(s) url generate depending of `NA.webconfig.httpSecure`, `NA.webconfig.urlHostname` and `NA.webconfig.urlPort`. This value does not contain `NA.webconfig.urlRelativeSubPath`.

staticNA.webconfig.variationsRelativePathString

Language and variable variation folder depending of languages.
Default Value:
  • "variations/".
